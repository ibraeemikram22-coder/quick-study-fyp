How to use Teacher Paper Generator

Build board-style exam papers from questions stored in the database.

1
Select filters

Choose class, board, exam type, and subject. These must match what was set up in the Admin panel.

2
Pick book & chapters

Click a book, then tick the chapters you want on the paper. Use Select All for a full-book test.

3
Check the pattern

The paper pattern (MCQ / short / long counts, marks, time) loads automatically for your board and exam type.

4
Generate paper

Click Generate Paper. Questions are picked randomly from the database. If you see “not enough questions,” add more in Admin.

Tip: Chapters and questions are added in the Question Bank Admin panel. Student Generate Paper is separate — it builds papers from uploaded notes, not the database.
