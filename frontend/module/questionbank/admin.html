                How to use Question Bank Admin

                Set up your syllabus once — teachers then generate papers from stored questions.

                1
                Setup tab — structure

                Add board, class, exam type, subject, book, and paper pattern (how many MCQs, short, and long questions per paper).

                2
                Chapters tab — content

                Select a book, enter a chapter title, paste lecture or textbook text, and save. This text is used to generate questions.

                3
                Generate tab — AI questions

                Pick a chapter and set counts. Click generate — questions are saved to the database automatically (about 30–60 seconds).

                4
                Review & import

                Use the Questions tab to view saved items. Import JSON if you have questions from an external tool. Teachers use their dashboard to build final papers.

                Tip: n8n is optional — built-in Gemini generation works without it. Load sample data on first run to see a working demo.
